## ইন্টারপ্রেটেড ল্যাঙ্গুয়েজ কি?
- একটি ইন্টারপ্রেটেড ল্যাঙ্গুয়েজ কোড এক লাইনে এক লাইনে রান করে, 
এবং এটি কম্পাইল করার প্রয়োজন হয় না।

### মূল বৈশিষ্ট্যসমূহ
1. সিঙ্গল-থ্রেডেড (Single-threaded):
* জাভাস্ক্রিপ্ট * একটি একক কল স্ট্যাক-এ চলে, অর্থাৎ এটি একসময় শুধু একটি কাজই করতে পারে।

2. ডিফল্টভাবে সিঙ্ক্রোনাস (Synchronous by default):
- জাভাস্ক্রিপ্ট কোড লাইন বাই লাইন, ক্রম অনুসারে এক্সিকিউট করে। প্রতিটি অপারেশন শেষ হওয়ার আগে পরবর্তী শুরু হতে পারে না।

3. অ্যাসিঙ্ক্রোনাস ক্ষমতা (Asynchronous capabilities):
- যদিও এটি সিঙ্গল-থ্রেডেড, জাভাস্ক্রিপ্ট ইভেন্ট লুপ (event loop) এবং ওয়েব API (Web APIs) ব্যবহার করে টাইমার, API রিকোয়েস্ট, এবং ইউজার ইন্টারঅ্যাকশন-এর মতো কাজগুলো মূল থ্রেড ব্লক না করে হ্যান্ডেল করতে পারে। এটি “মাল্টি-টাস্কিং” এর অনুভূতি দেয়।

4. মাল্টি-প্যারাডাইম (Multi-paradigm):
- এটি অবজেক্ট-ওরিয়েন্টেড (OOP), ফাংশনাল, এবং ইভেন্ট-ড্রিভেন প্রোগ্রামিং স্টাইল সমর্থন করে।

```bash
// অবজেক্ট-ওরিয়েন্টেড
class Person {
  constructor(name) { this.name = name; }
  greet() { console.log(`Hello, ${this.name}`); }
}
const p = new Person("Anik");
p.greet();

// ফাংশনাল
const add = (a, b) => a + b;
console.log(add(2, 3));

// ব্রাউজারে ইভেন্ট-ড্রিভেন উদাহরণ
const button = document.createElement('button');
button.textContent = 'Click me';
document.body.appendChild(button);

button.addEventListener('click', () => {
  alert('Hi there!');
});
```
#### প্রতিটি জায়গায় চলে (Runs everywhere):
- ব্রাউজারে (ক্লায়েন্ট-সাইড ইন্টারঅ্যাক্টিভিটির জন্য)
- সার্ভারে (Node.js ব্যবহার করে)
